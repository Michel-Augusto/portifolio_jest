# Portifolio Jest

Projeto com exemplos de teste de integração, API e contrato utilizando axios e jest.

### 📋 Pré-requisitos

npm 17.*+

### 🔧 Instalação

```
npm install -d
```

## ⚙️ Executando os testes

```
npm test
```

## 🛠️ Construído com

* [Jest](https://jestjs.io/)
* [Axios](https://axios-http.com/)
